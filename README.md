Works on Linux.

Prerequisites (Ubuntu-based):
sudo apt install build-essential libx11-dev libxft-dev fonts-dejavu

Compile:
gcc -Wall -Wextra -O2 -o macaua pb2.c -lX11 -lXft -lm -I/usr/include/freetype2
